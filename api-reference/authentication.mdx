# Authentication

All API requests must be authenticated using an API key provided in the `Authorization` header.

## Generating an API Key

Navigate to the dashboard to generate your API key.

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" https://api.decepticon.com/v1/projects
```
